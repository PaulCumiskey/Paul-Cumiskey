Phishing Analysis

Phishing Investigation: "Urgent Invoice"

Scenario: Analysis of a suspicious email reported by a user to the SOC mailbox.

Technical Breakdown

  • Source: Traced to 193.168.x.x (Russia-based IP).
  • Attachment: Invoice_992.zip containing an obfuscated JS downloader.
  • Findings: The script attempts to download a Stage-2 payload from a compromised WordPress site.

Outcome: Malicious domains blocked; user credentials reset.

SIEM / Detection

Splunk: Detecting RDP Brute Force

Objective: Create a high-fidelity alert for Event ID 4625 (Failed Login).

The Query

index=windows sourcetype="XmlWinEventLog:Security" EventCode=4625
| stats count BY TargetUserName, IpAddress
| where count > 15

Tuning: Filtered out service accounts to reduce false positives by 40%.

Network Forensics: Emotet C2 Beaconing

Tools: Wireshark, Zeek

Identified a 60-second "heartbeat" beacon to a suspicious port (8080) using I/O Graphs. Analyzed the HTTP GET request to identify the infected host and the specific malware family.
